# SM2SLK_BeatUp
Source code to convert .sm file type and .slk file for Beat-Up mode of Audition Dance Online game.

1. Structure of sm file this tool support:
   Beat-Up mode layout in ArrowVortex (thanks nongginga :D)
   1 : key 7 | 2 : key 4 | 3 : key 1 | 4 : key 9 | 5 : key 6 | 6 : key 3 | 7 : key space | 8 : finish move

2. How to using this tool?
    Download .py file, run cmd at place you put this tool, type py BeatUp8row_CLI.py, select file and you already have slk or sm file for Beat-Up mode.
    If you feel lazy to understand how this code work, so you can go to "Release" and download executable file to ussing that.
